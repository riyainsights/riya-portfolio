# Riya Yadav Portfolio

A modern, responsive, and professional personal portfolio website built for showcasing skills, projects, education, certifications, and contact information.

## Features

- Modern responsive design
- Smooth animations
- Typing text effect
- Professional UI/UX
- Contact form using EmailJS
- Mobile friendly
- SEO optimized

## Built With

- HTML5
- CSS3
- JavaScript (ES6)
- EmailJS
- AOS Animation
- Typed.js
- Font Awesome
